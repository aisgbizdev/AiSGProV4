üìä AGREGASI MARGIN & TIM
==================================================
Jika LEVEL = BC:
   Tidak memiliki bawahan. Margin = pribadi.

Jika LEVEL = SBC:
   Hitung BC di bawah tanggung jawab langsung.

Jika LEVEL = BSM:
   Hitung SBC + BC di bawah tim.

Jika LEVEL = SBM:
   Hitung BSM + SBC + BC di bawahnya.

Jika LEVEL = EM:
   Hitung SBM + BSM + SBC + BC di bawahnya.

Jika LEVEL = SEM:
   Hitung EM + SBM + BSM + SBC + BC di bawahnya.

Jika LEVEL = VBM:
   Hitung SEM + EM + SBM + BSM + SBC + BC di bawahnya.

Jika LEVEL = BM:
   Hitung semua di bawahnya (VBM hingga BC).

Jika LEVEL = CBO:
   Hitung seluruh struktur di bawah (BM hingga BC).

# Semua margin dijumlahkan berdasarkan anggota aktif di bawah fungsi pembinaannya,
# bukan hanya berdasarkan jabatan formal.
